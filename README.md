# Intellisia

## 🚀 실행 방법

### 로컬 환경에서 실행

1.  **의존성 설치:**
    ```bash
    # npm install
    ```

2.  **실행:**
    ```bash
    # npm run dev 
    ```

### Docker로 실행

1.  **Docker 이미지 빌드**
    ```bash
    # docker build -t dockerfile .
    ```

2.  **Docker 컨테이너 실행:**
    ```bash
    # docker run -d -p 80:3000 dockerfile
    ```

## 🌐 사이트 URL

-   **개발 환경:** `http://localhost:3000`
-   **배포 환경:** `http://installisa.site`
